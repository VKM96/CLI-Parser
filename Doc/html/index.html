CLI Interface 
This is a simple CLI interface developed for resource constrained environments.
The code is best suited for usage in embedded environments

Code Structure

  1. Source files are present in Src
  2. Interface files are present in inc
  3. PLease refer doxygen generated HTML documentaion in the Doc/html subfolder for implementation details
  4. The function entry point Main.c

Usage

  1. Initialization of the CLI is achieved through CliInit in cli_Interface.h
  2. At initialization the function for outputting to the console needs to be passed to CliInit.
  3. UserPrint is to be implemented based on the platform, as an example printf is used.

Input

  1. Function cmdParserInput needs to be invoked by the serial input to the system by the user.
  2. In the demo program here cmdParserInput is invoked by CliMockStream.
  3. The CLI process is handled by CliProcess in cli_Interface.h

Modification

  1. Update command table gCmdTble in Cli_Interface.c with command name and function.
  2. To change the buffer sizes update Macros in cmd_parser.h , the delimiter is changed my modifying variable DELIM in cmd_parser.c
